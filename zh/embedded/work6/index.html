用minicom发送文件到嵌入式平台

Send file

  • Set the file path which you should choose file from

    • Press Ctrl + A , then press O

    • Choose Filenames and paths

  • Set the A-Download directory (Download to PC from your device)

  • Set the B-Upload directory (Upload to your device from PC), you will choose the file you want to upload from the path.

  • Press esc and choose Save setup as dfl

  • In minicom(embedded device’s terminal)

rx filename
  • Press Ctrl + A , then press S
  • Choose xmodem
  • Choose your file

Run the file(Qt Gui)

chmod a+x filename
./filename

If you don’t have a desktop environment

Add -qws option:

./filename -qws

If you want transform the display

Add -display and your transform operation(Counterclockwise):

./filename -display "Transformed:Rot270"
